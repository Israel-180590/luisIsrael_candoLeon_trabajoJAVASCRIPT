## Descripción del Proyecto

Este proyecto corresponde al Ejercicio Final Obligatorio del módulo **JavaScript Avanzado**. 
Consiste en el desarrollo completo de un sitio web funcional utilizando HTML, CSS y JavaScript moderno, integrando AJAX, validaciones, cálculos dinámicos y un mapa interactivo. 

El objetivo es demostrar dominio de JavaScript aplicado a un proyecto real, organizado y funcional.

## Estructura del sitio web

- **index.html** – Página de inicio con 4 secciones y noticias cargadas mediante AJAX desde un archivo externo (`/data/noticias.json`). 
- **views/galeria.html** – Galería dinámica generada con JavaScript. 
- **views/presupuesto.html** – Formulario de presupuesto con validaciones, cálculo automático, extras y descuento por plazo. 
- **views/contacto.html** – Página de contacto con mapa dinámico implementado con Leaflet. ---
